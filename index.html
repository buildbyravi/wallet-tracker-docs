How to set up

  1. Get a copy of the Sheet

    Click the button above — it copies the Sheet and the script setup to your Google account.

  2. Add the library

    Extensions → Apps Script → Libraries (+) → paste the Script ID → identifier = WalletTracker → Add.

  3. Configure your RPC and wallets

    Row 1 col C+ = chain name · Row 2 col C+ = RPC URL · Row 3+ col B = wallet addresses (0x…).

  4. Fetch balances

    Balances → Fetch ALL chains — this sheet. Done. Balances appear in seconds.
